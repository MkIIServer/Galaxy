# oktw-world
讓每個玩家都可以擁有一個自己的世界

# Permission
* `oktw.command.world`
  * `oktw.command.world.create` 創建個人世界
  * `oktw.command.world.home` 回到個人世界
  * `oktw.command.world.info` 獲取世界資訊
  * `oktw.command.world.rebuild` 重建世界的指令
* `oktw.world`
  * `oktw.world.portal` 使用傳送門
  * `oktw.world.gamemode` 切換世界時保持Gamemode
